
This projects contains the sources for the talk "Reproducibility in Scientific Software Repositories and the Reusable Execution Environment" at deRSE 2026 in Stuttgart:

https://events.hifis.net/event/2945/contributions/21120/

The presentation is done in slidev, check the ./slidev directory for how to run it. I recommend using the .devcontainer with vscode to get an environment with node.
